# Project Governance

This document outlines the governance and maintainer structure for AnimeTV.

## Maintainer Roles

* **Core Maintainers**: Responsible for architectural direction, final pull request reviews, and security patches.
* **Contributors**: Active developers helping write features, fix scraping patterns, and add language translations.

## Release Policy

1. Releases occur on a monthly schedule.
2. Beta builds are published through GitHub Releases and Android testing channels.
3. Every release must pass build scripts on both Windows (Electron) and Android targets.
